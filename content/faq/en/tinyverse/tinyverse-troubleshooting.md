# Troubleshooting Tinyverse

Common issues with the Tinyverse dashboard and how to resolve them.

## Device shows as offline

- Verify your Tinyfarm is powered on and the LED is not blinking red (1.5 only)
- Check that your WiFi router is online
- Try power-cycling the Tinyfarm by unplugging and reconnecting USB
- If the device was recently re-flashed, it may take a minute to reconnect

## Charts not loading

- Try refreshing the page
- Check your internet connection
- Clear your browser cache if the issue persists
- Try a different browser (Chrome or Edge recommended)

## Data gaps in charts

Gaps in sensor data usually mean the device was temporarily offline. Common causes:

- Power interruption (USB disconnected)
- WiFi outage
- Router restart

The device will resume sending data automatically once connectivity is restored.

## Can't create a new experiment

- Make sure you have at least one device connected
- Check that the device you want to use isn't already assigned to another active experiment
- Try refreshing the page

## Login issues

- Use the **Forgot password** link on the login page to reset your password
- Make sure you're using the email you signed up with
- Check your spam folder for the verification or reset email
