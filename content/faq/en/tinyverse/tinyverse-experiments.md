# Managing Experiments

Experiments let you track growing sessions with your Tinyfarm.

## Creating an experiment

1. Go to **Experiments** in the sidebar
2. Click **New Experiment**
3. Give it a name (e.g. "Basil batch #3")
4. Select which Tinyfarm device to use
5. Set your target parameters (temperature, humidity, CO₂)
6. Click **Start**

## Monitoring progress

While an experiment is running, you can see:

- **Live readings** — current temperature, humidity, and CO₂ levels
- **Target vs actual** — how close your environment is to the set targets
- **Timeline** — a log of events and parameter changes

## Ending an experiment

Click **Stop Experiment** when your grow is complete. All data is saved and can be reviewed later under the experiment's history page.

## Tips

- Run one experiment per device for the clearest data
- Name experiments descriptively so you can compare runs later
- Check the timeline if readings look unexpected — it may show when conditions changed
