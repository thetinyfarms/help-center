# Data & Charts

Tinyverse stores all sensor data from your Tinyfarm devices and lets you visualize it.

## Viewing charts

1. Go to **Data** in the sidebar
2. Select a device and date range
3. Choose which sensors to display (temperature, humidity, CO₂, fluid level)

Charts update in real time when viewing live data.

## Exporting data

You can export your data as a CSV file:

1. Navigate to the data view for your device
2. Set the date range you want
3. Click **Export CSV**

The file will include timestamped readings for all selected sensors.

## Understanding the readings

| Sensor | Unit | Typical range |
|--------|------|---------------|
| Temperature | °C | 18–28 |
| Humidity | % | 40–80 |
| CO₂ | ppm | 400–2000 |
| Fluid level | % | 0–100 (Tinyfarm 1.5 only) |

## Data retention

All data is stored indefinitely. You can access historical readings from any past experiment or time period.
