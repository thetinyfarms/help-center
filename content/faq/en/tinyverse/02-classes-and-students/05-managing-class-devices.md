# Managing Class Devices

You can add devices to your class for organizational purposes. This helps you keep track of which devices are associated with each class.

### Adding devices to a class

1. Go to your class page
2. In the Devices section, click the **Add** button
3. Select an existing device you've already authenticated, or add a new device by entering its name and PIN
4. Click "Add to class"

The device is linked to your class and appears in the class devices list with its online/offline status.

📷 *Screenshot: Add device dialog showing list of available devices*

### Managing class devices

In the Devices section, you can:
- View all devices linked to your class with their online/offline status
- Search devices by name
- Remove devices from the class (click the three-dot menu)
- Refresh device status manually

📷 *Screenshot: Class devices list showing device status and actions*

> [!info]
> Adding a device to a class currently serves as an organizational tool for teachers. Students will still need to [authenticate devices individually in their experiments](#51-adding-device-to-experiment) to use them. We're planning to make class devices automatically available to all students in the class in a future update.

> [!tip]
> Device status updates automatically every 2 minutes, but you can click the refresh button to check status immediately.
