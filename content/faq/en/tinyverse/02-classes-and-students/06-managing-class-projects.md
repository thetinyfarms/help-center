# Managing Class Projects

Turn any of your experiments into a project that your students can access and use as a template. Students can create their own experiments based on your project.

### Adding a class project

1. Go to your class page
2. In the Class Projects section, click "Add project"
3. Select the experiment you want to turn into a class project
4. Click "Add as project"

The experiment is now available as a project for students in this class.

📷 *Screenshot: Class page showing Class Projects section with "Add project" button*

📷 *Screenshot: Dialog showing list of experiments to turn into class projects*

> [!tip]
> You can also turn experiments into class projects from your [My experiments page](#31-creating-new-experiment) (three-dot menu → "Turn into Class Project") or from within the experiment editor using the "Add class project" button.

> [!info]
> The same experiment can be a class project in multiple classes. Each class will see it independently in their Class projects section.

### Removing a class project

To remove a project from your class:

1. Go to your class page
2. In the Class Projects section, click the three-dot menu on the project
3. Select "Remove project"

Students will no longer see the project in their Class projects section.

📷 *Screenshot: Class Projects section showing project with dropdown menu and "Remove project" option*

> [!tip]
> You can also remove projects using the "Turn into Class Project" dialog from the experiments page or experiment editor by toggling off the classes.
