# How Students Join a Class

You don't need an email address or traditional account to use the tinyverse. Join using a Class ID (from your teacher), a username, and a simple 4-digit PIN — making it easy to get started.

### How to join

1. Go to the [join page](https://universe.tiny.science/student/join)
2. Enter the **Class ID** your teacher shared with you (6 characters)
3. Type your **username** (it will be converted to lowercase automatically)
4. Create a **4-digit PIN** (numbers only — you'll use this to log in later)
5. Click **Submit**

📷 *Screenshot: The student join page showing Class ID, username, and PIN input fields*

If you already have an account with the username and Class ID you introduced, the system will log you in automatically. If the username is already taken in your class, you'll see an error and need to choose a different one.

> [!tip]
> Your login information is saved in your browser, so you won't need to enter it every time. Just keep your PIN safe!

> [!info]
> If you forget your PIN, ask your teacher — they can [look it up or reset it](#04-managing-students) from the class management page.

Once you've joined, you'll be taken to your student dashboard where you can see class projects and start working on experiments.
