# Creating a Class

Classes help you organize your students and assign them experiments to work on. Each class gets a unique code (Class ID) that students use to join.

### How to create a class

1. Click the **+** button next to "Classes" in your sidebar
2. Enter a name for your class (e.g., "Biology Period 3" or "Grade 8 Science")
3. Click "Add class" or press Enter

Your new class is created immediately with a unique Class ID. You'll be taken to the class page where you can start adding students and managing the class.

📷 *Screenshot: Create class dialog with class name field*

> [!tip]
> Your class automatically inherits your language preference and school affiliation. This helps keep everything organized.

> [!info]
> Currently, each class can have only one teacher. We're planning to add support for multiple teachers per class, allowing teachers to join and collaborate on existing classes.
