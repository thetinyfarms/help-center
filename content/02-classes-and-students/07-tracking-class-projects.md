# Tracking Class Projects

After creating a class project, you can see which students have created experiments based on your template and view their work.

### Viewing student experiments

On your class page, the Class Projects section shows all your assigned projects. Each project displays the number of student experiments created from it.

Click on any class project to see:
- All student experiments created from that project
- Student usernames
- When each experiment was last updated
- Experiment thumbnails and descriptions

### Opening student experiments

From the project detail view, click "Open" on any student experiment to view it. You can see all their pages, modules, and data, but you cannot edit their work.

📷 *Screenshot: Student experiment opened in view-only mode*

> [!info]
> Student experiments are automatically linked to the original class project. When students create an experiment from your project, it appears in your tracking view immediately.

> [!tip]
> Check the "last updated" timestamp to see which students have been actively working on their experiments recently.
