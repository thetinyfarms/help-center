# Getting Started with Tinyfarm

Welcome to Tinyfarm! This guide will help you set up your device for the first time.

## What's in the box

- Tinyfarm device
- USB-C cable
- Quick start card

## First steps

1. **Plug in your Tinyfarm** using the included USB-C cable
2. **Flash the latest firmware** using the [firmware flash tool](/tinyfarm)
3. **Configure WiFi** so your device can connect to Tinyverse
4. **Open Tinyverse** at [tinyverse.live](https://tinyverse.live) to see your device
