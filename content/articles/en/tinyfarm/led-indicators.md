# LED Indicators (Tinyfarm 1.5)

The Tinyfarm 1.5 has an onboard LED status screen that shows device state at a glance.

## LED patterns

| Pattern | Meaning |
|---------|---------|
| Steady green | Connected and running normally |
| Blinking green | Sending data to Tinyverse |
| Steady blue | WiFi connected, waiting for first sync |
| Blinking blue | Connecting to WiFi |
| Steady red | Error — check Tinyverse for details |
| Blinking red | No WiFi connection |
| Off | Device is not powered |

## Tinyfarm 1.0

The original Tinyfarm 1.0 does not have an LED indicator. Use Tinyverse to check device status.
