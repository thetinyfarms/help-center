# Navigating the Dashboard

The tinyverse dashboard is designed to be intuitive and easy to navigate. The layout differs between teacher and student views, with each tailored to your specific needs.

## Sidebar navigation

### Teacher
- **Community:** Discover and duplicate public experiments shared by other teachers
- **My experiments:** Your personal experiment library
- **Classes:** View all your classes with a **+** button to create new ones. Each class appears as a clickable link below
- **Account (bottom):** Your name and school, with a dropdown to access Account settings and Logout.

📷 *Screenshot: Teacher sidebar showing navigation menu, classes list, and profile button*

### Student

- **Class info:** Shows your teacher's name, school, and a dropdown with your class code and copy button
- **Main navigation*** Class projects:** View class projects assigned by your teacher
- **My experiments:** Your personal experiment library
- **Account (bottom):** Your username with a dropdown to access Account settings and Logout. Next to that is a notification bell for group invitations and updates from classmates.

📷 *Screenshot: Student sidebar showing class info, navigation menu, profile button, and notification bell*

## Bottom-right corner

You'll always find quick-access buttons for:
- **Language selector:** Switch your interface language
- **Theme toggle:** Change between light and dark mode
- **Help menu (?):** Access help center, technical support, report bugs, and give feedback

📷 *Screenshot: Bottom-right corner showing language selector, theme toggle, and help menu buttons*

> [!tip]
> The sidebar stays in the same place across all pages, so you can always access navigation, your profile, and notifications (for students) no matter where you are in the platform.

> [!info]
> Students can copy their class code directly from the class dropdown in the sidebar to easily share it with classmates who want to join the same class.
